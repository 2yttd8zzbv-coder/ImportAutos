# ImportAutos 🚗

Calculadora de costos para importación de vehículos a Costa Rica.

## Características

- Cálculo automático de costos de importación
- Desglose detallado de todos los gastos
- Interfaz moderna y responsiva
- Visualización clara de impuestos y aranceles

## Instalación

```bash
npm install
npm run dev
```

## Cómo usar

1. Ingresa el valor del vehículo en dólares
2. La calculadora automáticamente sumará:
   - Flete: $1,375
   - Seguro: $100
   - Impuestos (65% de subtotal)
   - Otros gastos fijos
